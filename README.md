# logo-detect

## Install all the dependencies:
	>> pip install -r requirements.txt

## Run the server:
	>> python manage.py runserver 0.0.0.0:8000

## Endpoint for Image:
	>> /api/v1/logo_detect
## Endpoint for URL:
	>> /api/v1/logo_url_detect
## Endpoint for Training:
	>> /api/v1/train_model
## Endpoint for Update Dataset:
	>> /api/v1/update_dataset
